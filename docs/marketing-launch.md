# WizBlock Launch Copy

## One-Line Pitch

WizBlock is a free, GPL-3.0 open-source Android focus blocker with no ads, no account, no analytics SDK, no cloud sync, and no internet permission.

## Short Announcement

I released WizBlock, a privacy-first Android blocker for apps, websites, and
keywords. It is free, open source under GPL-3.0, and intentionally has no
internet permission, no ads, no login, and no analytics SDK. Rules, schedules,
usage counters, and recent blocked attempts stay on the device.

GitHub: https://github.com/kirentanna/WizBlock

For GitHub or website copy, it is reasonable to describe WizBlock as a
privacy-first Android alternative to commercial focus blockers such as AppBlock,
BlockSite, StayFocusd, Freedom, and similar app or website blockers. Avoid this
competitor wording in Google Play listing metadata.

## Longer Announcement

I built WizBlock because focus blockers tend to ask for powerful Android
permissions while also adding accounts, ads, subscriptions, analytics, or cloud
features. WizBlock takes the opposite approach: it uses the Android permissions
needed to block apps and browser distractions, but it keeps the data local and
does not request internet access.

What it does:

- Blocks apps, websites, and keywords.
- Supports named Blocklists, editable Games/Shopping/Social/Video starter lists, schedules, daily usage limits, and quick sessions.
- Includes Strict Mode for sessions that cannot be stopped early.
- Offers optional Device Admin uninstall protection only for Strict Mode.
- Stores rules and history locally on the device.
- Ships as GPL-3.0-only open source.

The privacy model is simple: no account, no ads, no analytics SDK, no cloud
sync, and no `android.permission.INTERNET`.

GitHub: https://github.com/kirentanna/WizBlock
